TikTok Animal Audition Effect

UX, Project Lead, Animator, Art Director

Overview

This is an innovation effect combining audio detection and face transformation. The goal of this effect was to inspire TikTok creators to create funny videos through unexpected animal results. I worked with engineers, interaction designers, and designers.

What I did

  • Worked with engineers to implement UX wireframes into interactive product.
  • Created multiple UX interaction flow options
  • Conducted internal A/B testing to determine best user experience
  • Created multiple UI animations based on A/B test results

Impact

  • Over +200 million views and posts on TikTok

  • Introduced new audio detection technology to TikTok app

Effect Experience Breakdown

Context

The Research and Development team approached us with a new technology that detected animal sounds from a user’s voice. They asked us to come up with a way to integrate the new technology into the app that both supported TikTok’s ecosystem and showcased their research.

I led a brainstorm with 16 designers and engineers during which my idea was chosen for production.

User Flow

UX Interaction Iterations

1. Decrease user choice fatigue:

The audio tech had an option to detect up to 6 different animals. One goal for our effect was to allow users the freedom to choose any of the 6 animals. Following this logic, I created an initial interaction that gave people all animal options after the wheelspin UI finished.

After conducting 10 internal user tests, we found that users were overwhelmed by the number of choices and had trouble deciding within the 4 seconds allotted. I then changed the UI interaction to prompt users with 2 choice options, decreasing friction from choice fatigue.

2. UI animation variations:

I created several different animation to displaying the wheelspinning UI that encouraged users to make animal sounds. We went with the 1st choice, as it supported the "wheel of fortune" form factor from our UI design the best.

Localizations

Countries have different languages and animal sensitivities. I adjusted 15+ variations for different country requirements.
